## 下载依赖
```
npm i

如果npm i执行失败，可以执行npm cache clear --force清除缓存，再执行npm i下载依赖
```
## 启动命令
```
yarn serve`
```

或者

```
npm run serve
```

## 访问路径

http://localhost:8080/
